# Preferencias Globales

## Comunicación
- Responder siempre en español
- Ser directo y concreto, sin relleno innecesario
- Cuando algo es ambiguo, preguntar antes de asumir

## Código
- TypeScript strict mode siempre
- Async/await sobre .then()
- Manejo explícito de errores (nunca silenciar un catch)
- Nombrar variables en inglés, comentarios en español

## Flujo de trabajo
- Tests obligatorios antes de marcar cualquier tarea como done
- Outputs estructurados en JSON o XML para facilitar automatización futura
- Al terminar una tarea, resumir qué se hizo y qué sigue
